Input

Physiological signals (e.g. heart rhythm, variability, basic motion)

Pre-processing

Noise filtering and normalization

Temporal alignment into short cadence windows

Cadence Layer

Detection of rhythm stability, transitions, and micro-disruptions

Focus on change over time, not absolute values

Pattern Context

Patterns mapped to qualitative states (e.g. steady, fragmented, recovering)

No labels, no diagnoses

Interpretive Cue

Soft signal to the user (word, tone, or minimal visual)

Suggests awareness, not action

User Response (optional)

User notices or reflects

No obligation, no feedback loop enforcement

Loop

System continues sensing without escalating interpretation
Primary Use Case

Alma supports individuals experiencing cognitive or emotional overload by offering gentle cues that help them notice shifts in their internal rhythm, without pushing correction or optimization.
